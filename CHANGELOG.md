## [0.1.0] - 2025-01-01

### Added
- Initial release of PizzaHub ordering website
- Menu page with pizza browsing and size selection
- Shopping cart with add, remove, and quantity management
- Cart persistence via localStorage
- Checkout page with customer information form
- Order submission via API with JSON file storage
- Order confirmation page
- Responsive design with Tailwind CSS
- Dark mode support with theme toggle
- Docker deployment with multi-stage builds
- Performance monitoring component
- Comprehensive test suite with Jest and React Testing Library
